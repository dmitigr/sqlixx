# Easy to use C++ API for SQLite

`sqlixx` - is a header-only C++ API for SQLite. Just include `sqlixx.hpp`.

**ATTENTION, this software is "alpha" quality, use it at your own risk!**

**ATTENTION, the API is a subject to change!**
